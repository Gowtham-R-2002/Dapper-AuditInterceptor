Features

Database Operations
  • INSERT operations with full entity capture using OUTPUT clause
  • UPDATE operations with before/after snapshots
  • DELETE operations with entity preservation
  • Advanced SQL parsing with T-SQL ScriptDom
  • Parameterized query handling and validation
Context Tracking
  • Framework-agnostic user identification
  • IP address and user agent capture
  • Extensible custom property support
  • Machine and process information
  • Timestamp and thread tracking
Storage Options
  • SQL Server database storage (default)
  • File-based logging with JSON serialization
  • Email notifications for critical events
  • External API integration support
  • Chained writers for multiple destinations
Integration & Flexibility
  • Framework-agnostic dependency injection
  • Microsoft.Extensions.Logging support
  • 4 different registration methods
  • Selective and conditional auditing
  • Custom context providers
Performance & Architecture
  • Minimal dependencies (no EF Core, no Dapper)
  • Lightweight and fast execution
  • Efficient SQL parsing and caching
  • Thread-safe operations
  • Async/await support throughout
Advanced Patterns
  • Chained audit writers
  • Selective auditing by table/operation
  • Conditional auditing by environment/role
  • Multi-tenant context support
  • Composite audit strategies
